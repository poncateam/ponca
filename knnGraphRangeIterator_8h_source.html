15 * - Use `Set = BitSet<MAX_NUMBER_OF_POINTS>` for the fastest search : Provides trivial insertion and search,
16 * with O(1) complexity at the expense of memory.
17 *
18 * - Use `Set = HashSet<Traits::MAX_RANGE_NEIGHBORS_SIZE>` for bigger data set : Best case complexity for insertion
19 * and search is O(1) and worst case is O(N) (depends on the given dataset and on the chosen hashing function).
20 */
